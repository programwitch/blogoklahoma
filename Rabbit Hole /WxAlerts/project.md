# Rabbit Hole Project: OKWxAlerts

A project to post Oklahoma Weather Alerts to Mastodon.  

- Javascript
- [National Weather Service / API Web Service](https://www.weather.gov/documentation/services-web-api)
- JQuery
- Bootstrap
